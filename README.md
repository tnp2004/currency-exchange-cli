## Currency exchange command line

### Available currency
- bath (Thailand)
- yen (Japan)
- dollar (UnitedStates)
- pound (England)
- euro (French)

### How to use
```
cargo run <amount> <source_currency> <target_currency>
```

**Usage example**
```
cargo run 100 bath yen
```

**Helping**
```
cargo run -- -h
```
